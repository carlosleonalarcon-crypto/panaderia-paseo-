# Panadería Pastelería Paseo

Sitio web estático de **Panadería Pastelería Paseo** (Cartagena): pan artesanal, pastelería, encargos y contacto.

## Estructura

- `index.html` — página principal
- `css/` — estilos (`style.css` principal, `legal.css` páginas legales)
- `js/` — `main.js` (interacción, AOS, menú, FAQ) y `cookies.js` (banner RGPD)
- `assets/img/` — imágenes (logo, favicon, etc.)
- `legal/` — aviso legal, privacidad y cookies

## Despliegue

Proyecto listo para **GitHub** y **Vercel**: configurar el directorio raíz del proyecto como **`panaderia-paseo`** (o desplegar el contenido de esta carpeta como raíz del sitio).

Todas las rutas internas son **relativas** (sin `/` inicial).

## Dominio y SEO

- Dominio de referencia: `https://www.panaderiapaseo.es/`
- `sitemap.xml` en la raíz de esta carpeta
- Páginas legales con `noindex` (no incluidas en el sitemap)
